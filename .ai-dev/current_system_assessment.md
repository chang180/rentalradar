## 🏗️ 技術架構

### 後端堆疊
- **框架**: Laravel 12 + PHP 8.4
- **資料庫**: SQLite (開發) + MySQL (生產)
- **快取**: Redis 智慧分層策略
- **認證**: Laravel Fortify
- **開發環境**: Laravel Herd

### 前端堆疊
- **框架**: React + Inertia.js
- **地圖**: Leaflet.js + AI 優化
- **圖表**: Chart.js
- **樣式**: Tailwind CSS v4

## 🔧 開發環境

### 設定需求
```bash
# 安裝依賴
composer install
npm install

# 環境設定
cp .env.example .env
php artisan key:generate

# 資料庫設定
php artisan migrate
php artisan statistics:populate

# 開發伺服器
php artisan serve
npm run dev
```

### 關鍵指令
```bash
# 處理政府資料
php artisan rental:process

# 填充統計資料
php artisan statistics:populate

# 執行測試
php artisan test

# 程式碼格式化
vendor/bin/pint --dirty
```
